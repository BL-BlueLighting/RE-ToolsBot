RE-ToolsBot

使用 OneBot V11 + NoneBot 的 Nonebot 实例工具集合

RE-ToolsBot 是一个以 **OneBot V11** 和 **NoneBot** 为基础的 Nonebot 机器人，提供多种实用插件、配置管理与扩展能力。

快速上手

以下是基本使用流程：

  1. 克隆仓库：git clone https://github.com/BL-BlueLighting/RE-ToolsBot.git
  2. 使用 nonebot-cli 创建项目（或在已有 nonebot 项目里接入）：nb create
  4. 配置 `data/configuration_template.json` → 重命名为 `data/configuration.json` 并填入对应 API Key 等
  5. nb run --reload

更多细节请参考 项目仓库 的 README。

项目结构概览

下面是项目目录中比较重要的文件或文件夹：

  • bot.py — 项目入口脚本
  • data/ — 存放配置文件（如 API Key、json 配置）
  • toolsbot/plugins/ — 各种插件模块
  • setting/ — 全局配置、帮助脚本等
  • templates/ — （如有）用于 Web 输出或模板的目录
  • .env.dev / .env.prod — 环境变量配置文件
  • README.md / README.zh-cn.md — 项目说明文档
